### **1. 맵 생성을 위한 준비하기**

#### **1.1. 터미널 실행**

맵 생성을 하기 위해 터미널에서 우분투 창을 여러 개 띄워 실행시키도록 합니다.

**<Terminal 1>**

```bash
ros2 launch ydlidar_ros2_driver ydlidar_launch.py
```

**<Terminal 2>**

```bash
ros2 run tf2_ros static_transform_publisher 0 0 0 0 0 0 odom base_link
```

**<Terminal 3>**

```bash
ros2 launch slam_toolbox online_async_launch.py
```

**<Terminal 4>**

```bash
ros2 run rviz2 rviz2
```



### **2. rviz 실행**

#### **2.1. Map 디스플레이 설정 및 추가**

1. 추가: 왼쪽 하단 `Add` 클릭 → By topic → /map 선택하여 Map 디스플레이 추가
2. Qos 설정:
   - `Map` 항목을 펼친 후 Topic 섹션 → `Durability Policy`를 `Transient Local`로 변경

#### **2.2. 레이저 스캔(LaserScan) 설정 및 추가**

1. 추가: 왼쪽 하단 `Add` 클릭 → By topic → /scan 선택하여 LaserScan 디스플레이 추가
2. QoS 설정:
   - `LaserScan` 항목을 펼쳐서 Topic 섹션 → `Reliability Policy`를 `Best Effort`로 변경
